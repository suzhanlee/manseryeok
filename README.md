# manseryeok


### 사용법

1. uploadManSeryeokData api를 사용해 만세력 파일을 db에 저장한다.
2. uploadLuckPillarsData api를 사용해 대운 파일을 db에 저장한다.
3. createDestiny api를 사용해 사용자의 사주팔자를 본다.
```angular2html

    "birthDate": "1984-05-28",
    "calendarType": "양력",
    "gender": "woman",
    "name": "이진수",
    "birthTime": "06:47"
}
```
```angular2html
{
    "ganji": "甲子年 己巳月 壬戌日",
    "hourPillar": "癸卯"
}
```
4. createLuckPillars api를 사용해 사용자의 대운을 본다.
```angular2html
{
    "fourPillars": "甲子己巳壬戌癸卯",
```
